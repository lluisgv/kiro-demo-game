# Product

Kiro Soccer is a browser-based single-player soccer game built as part of an AWS Re:Invent workshop. The player controls a Kiro-logo sprite and kicks a ball into a goal on the right side of the screen. The game tracks goals scored and features a start screen, gameplay loop, and game over screen.

The project is used as a hands-on demo for building games with AI assistance (Kiro IDE). Participants extend and modify the game during the workshop session.
